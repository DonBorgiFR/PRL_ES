# Convenciones Editoriales y de Trazabilidad

## Principios

1. Cada guia debe citar norma y articulo origen.
2. Cada obligacion debe tener evidencia documental asociada.
3. Cada plantilla debe definir responsable y fecha de cierre.
4. El lenguaje debe ser tecnico pero operativo.

## Formato obligatorio por flujo

1. Objetivo operativo.
2. Base legal minima.
3. Disparadores de ejecucion.
4. Checklist de campo.
5. Evidencias obligatorias.
6. Criterio de cierre.
7. KPI de seguimiento.

## Convencion de IDs

- Flujo: FLW-XX
- Plantilla: TPL-XX-YY
- Evidencia: EVD-XX-YY

Ejemplo:

- FLW-01: Evaluacion de riesgos por puesto.
- TPL-01-01: Matriz de evaluacion inicial.
- EVD-01-01: Acta de validacion por responsable.

## Control de cambios

Registrar al final de cada documento:

- Fecha.
- Autor.
- Motivo.
- Norma impactada.

## Semaforo de cumplimiento

- Verde: evidencia completa y vigente.
- Ambar: evidencia incompleta o vencida.
- Rojo: sin evidencia o incumplimiento.
