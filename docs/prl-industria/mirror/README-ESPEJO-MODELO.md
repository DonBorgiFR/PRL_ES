# Modelo de README para el Repo Espejo

## Proposito

Repositorio documental para ejecucion PRL en industria, basado en obligacion legal y evidencia operativa.

## Uso rapido

1. Elegir flujo aplicable.
2. Ejecutar checklist.
3. Cerrar plantilla con firmas y fecha.
4. Reportar estado semanal en tablero.
