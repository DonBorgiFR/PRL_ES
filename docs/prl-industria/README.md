# PRL Industria - MVP Operativo

Este repositorio documental ayuda a ejecutar cumplimiento PRL en entorno industrial con foco rapido y accionable.

## Objetivo

Pasar de la norma a evidencia verificable en campo:

1. Obligacion legal.
2. Procedimiento operativo.
3. Registro firmado.
4. Seguimiento y cierre.

## Alcance MVP (6 flujos)

1. Evaluacion de riesgos por puesto y plan correctivo.
2. Induccion/formacion de nuevo ingreso con evidencia.
3. Coordinacion CAE para contratas y subcontratas.
4. Vigilancia de la salud y adaptaciones.
5. Registro de incidentes y cierre de acciones.
6. Condiciones del lugar de trabajo — verificacion RD 486/1997 (iluminacion, temperatura, ruido, carga manual, equipos).

## Normativa base usada en este MVP

- Ley 31/1995 (LPRL).
- RD 39/1997 (RSP).
- RD 171/2004 (CAE).
- RD 1627/1997 (Construccion, aplicable cuando haya obra).
- RD 486/1997 (condiciones de seguridad y salud en lugares de trabajo).

## Regla de uso rapido

Cada flujo se ejecuta en tres pasos:

1. Preparar (5-15 min).
2. Ejecutar checklist (15-40 min).
3. Cerrar evidencia (5-10 min).
